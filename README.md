# Embedded_SmartHome
## 스마트 홈을 위한 라즈베리 파이 기반 프로그램

----
## 개발 목적

임베디드 시스템을 이용해 스마트홈을 제작, 집의 기기들을 자동/원격으로 관리하는 것이 목적이다.
또한, 단순 기기 제어뿐만이 아닌 여가시간에 즐길 수 있는 게임 및 전기요금을 조회하는 기능도 부가적으로 탑재되어 있다.
